# Hospital Emergency Room Dashboard (Excel)

## Project Overview
This project is a dynamic Excel dashboard designed to analyze hospital emergency room data on a monthly basis.

## Key Metrics
- Total number of patients
- Average waiting time
- Patient satisfaction score
- Admission status
- Gender-wise analysis
- Age group analysis
- Department referrals

## Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers (Month & Year)
- Conditional Formatting

## Features
- Interactive slicers for year and month
- Dynamic KPIs and charts
- Automated insights based on filters

## How to Use
1. Download the Excel file
2. Open it in Microsoft Excel
3. Use slicers to interact with the dashboard
